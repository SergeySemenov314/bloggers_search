# Поиск блогеров для бартера

Инструмент, который сам закрывает цикл подбора блогеров под бартер для бренда
женской одежды (Wildberries / Ozon): анализирует базу «идеальных» блогеров,
находит похожих и пишет персональные офферы. Веб-версия: бэкенд на Python
(FastAPI), фронтенд на React. Действия агентов видны в интерфейсе живым трейсом.

## Статус

- **Этап 1 — Анализ базы** ✅ готов
- Этап 2 — Поиск новых — в работе
- Этап 3 — Персональные офферы — в работе

## Схема автоматизации (этап 1)

```
Google-таблица (CSV-экспорт)
        │  services/sheet.py — тянет ники из ссылок, чинит битые/нестандартные
        ▼
instaloader (без ключей)
        │  services/instagram.py — bio, подписчики, посты, имя, ссылка
        │  недоступен/битый → помечаем в трейсе, идём дальше
        ▼
Claude API (один вызов, дешёвая модель)
        │  agents/analyze.py + prompts.py — синтез «портрета идеального блогера»
        ▼
SSE-трейс → React-интерфейс (ход работы + итог + стоимость прогона)
```

Стоимость прогона: ~$0.02 на Haiku 4.5, ~$0.06 на Sonnet 5 (сбор данных
бесплатный, Claude тратится только на синтез портрета).

## Запуск

### Бэкенд (Anaconda Prompt)

```bat
cd D:\My_projects\ML_projects\bloggers_search\backend
pip install -r requirements.txt
copy .env.example .env
:: откройте .env и вставьте свой ANTHROPIC_API_KEY
uvicorn app:app --port 8080
```

### Фронтенд (нужен Node.js)

```bat
cd D:\My_projects\ML_projects\bloggers_search\frontend
npm install
npm run dev
```

Откройте http://localhost:5173 и нажмите «Запустить анализ».
